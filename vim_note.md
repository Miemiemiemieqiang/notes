
# Vim Keymap
## Move
`<C-f>` forward page

`<C-b>` backward page

`<C-d>` down page

`<C-u>` up page

## Copy
### Copy to clipboard
`"+p` copy to vim

`"+yy` copy current line to clipboard

## Caps
### Toggle caps
`~`
### Change caps
`gU`
change word upper caps
`gu`
change word lower caps

## Auto crease
### Auto increase
`<C-a>`

### Auto decrease
`<C-x>`
